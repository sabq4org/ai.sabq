# دليل تغطية البث الحي — الاعتداءات على دول الخليج

## الهدف من الصفحة

صفحة "البث الحي" هي صفحة تغطية لحظية مباشرة لجميع الاعتداءات والأحداث العسكرية التي تستهدف دول الخليج العربي. تعمل كـ "تايم لاين" مرتّب زمنياً يُحدَّث لحظياً، ويظهر تلقائياً على الصفحة الرئيسية للموقع.

**الرابط العام:** `/gulf-live`
**لوحة التحكم:** `/dashboard/gulf-events`

---

## خطوات إضافة حدث جديد

### 1. الدخول إلى لوحة التحكم
- ادخل على `/dashboard/gulf-events`
- يجب أن يكون لديك صلاحية **محرر** أو **مسؤول**

### 2. اضغط "حدث جديد"
- اضغط زر **"حدث جديد"** في أعلى الصفحة

### 3. اختر الدولة المستهدفة
اختر الدولة التي وقع فيها الحدث:
- السعودية
- الإمارات
- البحرين
- الكويت
- قطر
- عُمان
- اليمن

### 4. اختر نوع الحدث
هذا أهم حقل — اختر النوع المناسب بدقة:

| النوع | متى تستخدمه |
|-------|-------------|
| **صد مسيّرة** | عند اعتراض طائرة مسيّرة (درون) فقط |
| **صد صاروخ باليستي** | عند اعتراض صاروخ باليستي فقط |
| **صد صاروخ كروز** | عند اعتراض صاروخ كروز فقط |
| **صد صاروخ باليستي ومسيّرة** | عند اعتراض صواريخ باليستية ومسيّرات معاً في نفس الهجمة |
| **سقوط شظايا** | عند سقوط شظايا من الاعتراض |
| **لا أضرار** | عند تأكيد عدم وجود أضرار |
| **إصابات** | عند وقوع إصابات بشرية |
| **استشهاد** | عند وقوع حالات استشهاد |
| **بيان رسمي** | بيانات وزارات الدفاع والجهات الرسمية |
| **تصريح مسؤول** | تصريحات المسؤولين والمتحدثين الرسميين |
| **تحرك عسكري** | عمليات عسكرية أو ضربات |
| **إدانة دولية** | بيانات إدانة من دول أو منظمات دولية |

**مثال:** إذا أعلنت وزارة الدفاع عن "اعتراض 3 صواريخ باليستية و7 مسيّرات" ← اختر **"صد صاروخ باليستي ومسيّرة"**

**مثال:** إذا أعلنت عن "اعتراض وتدمير مسيّرة في المنطقة الشرقية" ← اختر **"صد مسيّرة"**

### 5. اختر المصدر
- **بيان رسمي** — بيانات وزارات الدفاع
- **وكالة أنباء رسمية** — واس، كونا، بنا، قنا
- **مراسل سبق** — تغطية مراسلي سبق الميدانية
- **وكالات دولية** — رويترز، فرانس برس
- **مصادر مطلعة** — مصادر غير رسمية
- **أخرى** — (يظهر حقل لكتابة اسم المصدر)

### 6. اختر مستوى الأهمية
- **عاجل** — أحداث كبرى تستدعي تنبيه فوري (استشهاد، هجوم واسع)
- **مهم** — أحداث مهمة لكن ليست طارئة (إصابات، شظايا)
- **عادي** — أحداث روتينية (اعتراض ناجح، بيان رسمي)

### 7. اكتب نص الخبر
اكتب نص الخبر بشكل مختصر وواضح.

**أمثلة جيدة:**
- "وزارة الدفاع: اعتراض وتدمير مسيّرة في الربع الخالي متجهة إلى حقل شيبة"
- "الدفاع القطرية: القوات المسلحة تتصدى لهجمة صاروخية استهدفت البلاد"
- "الصحة الكويتية: إصابتان طفيفتان تزامناً مع استهداف مبنى سكني بالمنقف"

### 8. وقت النشر (اختياري)
- **اتركه فارغاً** = ينشر بالوقت الحالي
- **حدد وقتاً** = لتأريخ الحدث بوقته الفعلي (مثلاً إذا كنت تضيف حدثاً وقع قبل ساعة)

### 9. انشر
- اضغط **"نشر فوري"** للنشر مباشرة
- أو اضغط **"حفظ مسودة"** للمراجعة لاحقاً

## خصائص إضافية

### تثبيت حدث
- اضغط أيقونة الدبوس بجانب أي حدث لتثبيته في أعلى القائمة

### إضافة تحديث على حدث
- اضغط أيقونة الربط بجانب الحدث لإضافة تحديث مرتبط به

### تعديل حدث
- اضغط أيقونة القلم لتعديل أي حدث منشور

### حذف حدث
- اضغط أيقونة سلة المهملات ← تأكيد الحذف
- الحدث يُنقل للأرشيف ولا يُحذف نهائياً

---

## ملاحظات مهمة

- الأحداث تظهر **فوراً** على الموقع وعلى الصفحة الرئيسية بمجرد النشر
- التغطية تعمل بتقنية **البث المباشر** — أي زائر على الصفحة يرى التحديثات لحظياً بدون تحديث الصفحة
- **الدقة مهمة** — اختاروا النوع الصحيح لأنه يظهر كتصنيف ملوّن بجانب كل حدث
- سجل النشاط يحفظ كل عملية إضافة/تعديل/حذف مع اسم المحرر
