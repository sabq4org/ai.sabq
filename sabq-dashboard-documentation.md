## نظرة عامة

لوحة تحكم سبق الذكية هي منصة إدارة محتوى متقدمة مدعومة بالذكاء الاصطناعي، مصممة لإدارة موقع سبق الإخباري بكفاءة عالية. تدعم اللوحة ثلاث لغات (العربية، الإنجليزية، الأوردو) وتوفر أدوات شاملة لإنشاء المحتوى ونشره وتحليله وتوزيعه عبر جميع القنوات الرقمية.

### الإحصائيات الرئيسية
| البند | التفاصيل |
|-------|----------|
| اللغات المدعومة | العربية، الإنجليزية، الأوردو |
| عدد نماذج الذكاء الاصطناعي | GPT-5.1, Claude Sonnet 4.5, Gemini 2.5 Flash |
| منصات الإشعارات | FCM (Android), APNs (iOS), Email, WhatsApp |
| التخزين السحابي | Google Cloud Storage, Cloudflare R2, Cloudflare Images |
| بوابات الدفع | Tap Payment Gateway |
| عدد الأدوار | 8 أدوار (من مدير النظام إلى القارئ) |

## المميزات الأساسية

### 1. نشر الأخبار والمقالات
- إنشاء وتحرير المقالات باستخدام محرر WYSIWYG متقدم
- دعم كامل للمحتوى ثلاثي اللغة (عربي، إنجليزي، أوردو)
- نشر فوري أو مجدول بتاريخ ووقت محددين
- تحويل المقال إلى "عاجل" بنقرة واحدة
- تصنيف تلقائي بالذكاء الاصطناعي
- توليد عناوين SEO محسنة تلقائياً

### 2. إدارة الأقسام والتصنيفات
- إنشاء وتعديل الأقسام الرئيسية والفرعية
- إعادة ترتيب الأقسام بالسحب والإفلات
- تخصيص ألوان وأيقونات لكل قسم
- تعيين slug مخصص لكل قسم

### 3. إدارة التعليقات
- لوحة إشراف شاملة للتعليقات
- فلترة تلقائية بالذكاء الاصطناعي للتعليقات المسيئة
- كشف الكلمات المشبوهة تلقائياً
- نظام حظر المستخدمين المخالفين
- تحليل المشاعر (إيجابي/سلبي/محايد) للتعليقات

## إدارة المحتوى

### محرر المقالات المتقدم
محرر نصوص غني يدعم:
- تنسيق النصوص الكامل (عناوين، قوائم، اقتباسات، جداول)
- إدراج الصور والفيديوهات ومعارض الصور
- إدراج تغريدات ومنشورات وسائل التواصل الاجتماعي
- معاينة فورية للمقال قبل النشر
- حفظ تلقائي كمسودة

### دورة حياة المقال
```
مسودة → قيد المراجعة → معتمد → منشور → مؤرشف
```

### خيارات النشر
| الخيار | الوصف |
|--------|-------|
| النشر الفوري | نشر المقال مباشرة |
| النشر المجدول | تحديد تاريخ ووقت النشر |
| النشر كعاجل | تفعيل شريط الأخبار العاجلة |
| إعادة النشر | إعادة نشر مقال سابق بتحديثات |

### نظام الترجمة الذكي
- ترجمة فورية بنقرة واحدة من العربية إلى الإنجليزية
- ترجمة مع الحفاظ على أسلوب سبق التحريري
- نشر تلقائي للنسخة المترجمة
- ربط النسخ اللغوية ببعضها (hreflang)

## نظام الذكاء الاصطناعي (iFox)

### لوحة iFox الرئيسية
بوابة ذكاء اصطناعي متكاملة تضم جميع أدوات AI في مكان واحد.

### 1. مولّد المحتوى الذكي (Sabq Editorial AI)
- تحليل وتنقية المحتوى الوارد تلقائياً
- إزالة التوقيعات والعناوين والبيانات الوصفية
- إعادة صياغة بأسلوب سبق المهني
- دعم ثلاث لغات: العربية، الإنجليزية، الأوردو
- ضبط النبرة (مهني، موضوعي)
- تنسيق HTML تلقائي

### 2. توليد العناوين الذكية
- مقارنة عناوين متعددة محسنة لمعدل النقر (CTR)
- تحسين SEO تلقائي للعناوين
- اقتراح عناوين بديلة مع تقييم الجودة

### 3. توليد منشورات التواصل الاجتماعي
- إنشاء منشورات مخصصة لـ Twitter/X, Facebook, LinkedIn
- اقتراح هاشتاقات تلقائية
- تخصيص المحتوى حسب كل منصة

### 4. توليد الصور بالذكاء الاصطناعي
- توليد صور باستخدام Gemini 2.5 Flash Image
- بحث ذكي عن صور مخزنة تناسب المقال
- كشف النقطة البؤرية (Focal Point) تلقائياً
- توليد نص بديل (Alt Text) للصور

### 5. التصنيف الذكي
- تصنيف تلقائي للمقالات في الأقسام المناسبة
- تطبيق "القاعدة الذهبية": الأخبار ذات الفاعل السعودي = أخبار محلية
- تقييم جودة المحتوى (0-100 نقطة)
- تحليل الاتجاهات الإخبارية الحالية

### 6. التلخيص الذكي
- تلخيص المقالات الطويلة إلى ~30% من الحجم الأصلي
- الحفاظ على الحقائق والأرقام الأساسية
- استخدام Claude Sonnet 4.5 للتلخيص العربي

### 7. التحقق من الحقائق
- أداة مدمجة لكشف المعلومات المضللة
- التحقق من الادعاءات والأرقام
- تصنيف مستوى المصداقية

### 8. إنفوجرافيك البيانات
- توليد رسوم بيانية تفاعلية من محتوى المقال
- مكونات متنوعة: أرقام كبيرة، مخططات دائرية، جداول زمنية
- تحسين للعرض بالعربية (RTL)

### 9. إدارة مهام الذكاء الاصطناعي
- جدولة مهام AI في الخلفية
- تنفيذ يدوي أو تلقائي
- متابعة حالة كل مهمة
- تنظيف تلقائي للمهام المكتملة

### 10. إعدادات الذكاء الاصطناعي
- تخصيص تفضيلات AI (النبرة، الأسلوب، اللغة)
- إدارة قوالب المحتوى
- تعريف قواعد سير العمل الآلية
- فحص جودة المحتوى الآلي

## إدارة الوسائط والمكتبة الإعلامية

### المكتبة الإعلامية المركزية
- عرض شبكي وقائمة للملفات
- بحث بالاسم والنوع والمجلد
- تنظيم هرمي بالمجلدات
- تصفية حسب التصنيف والمفضلة

### رفع الملفات
| النوع | الحد الأقصى | الصيغ المدعومة |
|-------|-------------|---------------|
| الصور | 10 ميجابايت | PNG, JPG, JPEG, WebP, GIF |
| الفيديو | 10 ميجابايت | MP4, WebM |
| المستندات | 10 ميجابايت | PDF, DOCX |

### تحسين الصور التلقائي
- تحويل تلقائي إلى WebP/AVIF
- توليد أحجام متجاوبة (320, 640, 960, 1280 بكسل)
- توليد صور مصغرة تلقائياً
- إنشاء placeholder ضبابي للتحميل البطيء
- تحسين عبر Cloudflare CDN على الحافة

### التخزين السحابي
- **Google Cloud Storage (GCS)**: التخزين الأساسي
- **Cloudflare R2**: تخزين بديل عالي الأداء
- **Cloudflare Images**: تحسين وتوزيع الصور عبر شبكة CDN عالمية
- فصل بين الملفات العامة والخاصة (public/private)
- وصول آمن عبر وكيل للملفات الخاصة

### بيانات الوسائط الوصفية
- تتبع اسم الملف الأصلي، الحجم، نوع MIME
- أبعاد الصورة (عرض وارتفاع)
- نص بديل (Alt Text) - يُولّد بالذكاء الاصطناعي
- نقطة بؤرية (Focal Point) للاقتصاص الذكي

## التحليلات والتقارير

### 1. لوحة التحليلات الرئيسية
- إحصائيات المقالات: المشاهدات، الإعجابات، الحفظ، المشاركات، التعليقات
- مقاييس القراءة: متوسط وقت القراءة، عمق التمرير، معدل الإكمال
- إحصائيات اليوم: أخبار اليوم، الأكثر مشاهدة، متوسط المشاهدات

### 2. تحليلات المقالات التفصيلية
- بحث وفلترة شامل لأداء المقالات
- بطاقات إحصائية: المشاهدات، الإعجابات، الحفظ، المشاركات
- خريطة حرارية للتفاعل (Engagement Heatmap)
- تصدير التقارير كملفات PDF

### 3. متتبع الزوار المباشر
- عدد الزوار النشطين في الوقت الحقيقي
- الصفحات الأكثر زيارة حالياً
- نبض القلب (Heartbeat) لتتبع الجلسات الحية
- عداد المشرفين المتصلين حالياً

### 4. تحليلات النشرات الإخبارية
- معدل فتح الرسائل ومعدل النقر
- اتجاهات الأداء عبر الزمن
- تتبع الأحداث: فتح البريد والنقر على الروابط

### 5. تحليلات الإعلانات
- مؤشرات الأداء: الظهور، النقرات، التحويلات، CTR, CPC, CPM
- تقسيم الجمهور حسب الجهاز والبلد والمرجع
- تتبع مباشر في الوقت الحقيقي
- تصدير البيانات كـ CSV

### 6. تحليلات التخصيص
- أداء التوصيات الذكية (ظهور مقابل نقرات)
- توزيع اهتمامات المستخدمين
- تحليل شرائح المستخدمين

### 7. تحليلات المشاعر
- تحليل مشاعر التعليقات بالذكاء الاصطناعي (إيجابي/محايد/سلبي)
- اتجاهات المشاعر عبر الزمن
- أكثر المقالات تفاعلاً من حيث المشاعر

### 8. ساعات الذروة
- خريطة حرارية لساعات الذروة عبر أيام الأسبوع
- تحديد أفضل أوقات النشر

### 9. إنتاجية فريق العمل
- تتبع عدد المقالات لكل محرر/مراسل
- مقاييس التفاعل لكل كاتب

## نظام الإشعارات والتنبيهات

### إشعارات الدفع (Push Notifications)

#### iOS (APNs)
- اتصال HTTP/2 مباشر مع خوادم Apple
- مصادقة JWT
- دعم الإشعارات الغنية (صور وأصوات مخصصة)
- صوت خاص للأخبار العاجلة (`breaking.caf`)
- مستويات أولوية: عادي، حساس للوقت (time-sensitive)

#### Android (FCM)
- Firebase Cloud Messaging V1 API
- إرسال جماعي (Batch Sending)
- تسجيل الأجهزة تلقائياً

### إدارة الحملات
- إنشاء حملات إشعارات مستهدفة
- تقسيم الجمهور (Segments) حسب:
  - نوع الجهاز (iOS/Android)
  - الاهتمامات والأقسام المفضلة
  - مستوى النشاط
- جدولة الحملات بتاريخ ووقت
- تحليلات الحملة: الفتح والنقرات

### الإشعارات الفورية (SSE)
- إشعارات في الوقت الحقيقي عبر Server-Sent Events
- صندوق إشعارات لكل مستخدم
- تخصيص تفضيلات الإشعارات

### إرسال سريع
- إرسال إشعار فوري لمقال بنقرة واحدة
- ربط تلقائي بالمقال المنشور

## النشرات الإخبارية والبريد الإلكتروني

### النشرات الذكية (Smart Newsletters)
- **مزود الخدمة**: MailerLite لإدارة المشتركين
- **أنواع النشرات**:
  | النوع | الوصف |
  |-------|-------|
  | `morning_brief` | ملخص الصباح |
  | `evening_digest` | ملخص المساء |
  | `weekly_roundup` | التقرير الأسبوعي |
  | `breaking_news` | الأخبار العاجلة |
  | `personalized_digest` | ملخص مخصص حسب اهتمامات المستخدم |

- **مميزات القوالب**:
  - تصميم متوافق مع العربية (RTL)
  - عناصر تفاعلية: سؤال اليوم، إحصائيات سريعة
  - محتوى حصري ومحتوى صوتي
  - تزامن ثنائي الاتجاه مع MailerLite

### النشرة الصوتية (Audio Newsletter)
- **محرك الصوت**: ElevenLabs
- **الأصوات المتاحة**:
  | الصوت | النوع |
  |-------|-------|
  | فارس | ذكر - أخبار |
  | رائد | ذكر - سعودي |
  | أبرار صباح | أنثى - أخبار |
- توليد نصوص البرنامج تلقائياً (مقدمة - فقرات - انتقالات - خاتمة)
- تخزين على R2/Object Storage
- خلاصة RSS متوافقة مع iTunes/Podcasts

### وكيل البريد الإلكتروني (Email Agent)
- **استقبال**: عبر SendGrid Inbound Parse
- **المعالجة الآلية**:
  - استلام بريد MIME وتحليله
  - منع التكرار عبر قاعدة بيانات التتبع
  - استخراج المحتوى من: Word (.docx), PDF, صور (OCR بالذكاء الاصطناعي)
  - استخراج الصور المضمنة ورفعها تلقائياً
  - تطبيق أسلوب سبق التحريري بالذكاء الاصطناعي
  - تصنيف تلقائي في القسم المناسب

### إدارة قوالب البريد
- تصميم وتعديل قوالب البريد الإلكتروني
- معاينة فورية قبل الإرسال
- متغيرات ديناميكية للمحتوى المخصص

## نظام واتساب الذكي

### التكامل مع Twilio
- **المزود**: Twilio WhatsApp Business API
- **المصادقة**: رمز أمان (Token) من 64 حرفاً أو بصيغة `SABQ-XXXX`

### الأوامر المدعومة
| الأمر | الوظيفة |
|-------|---------|
| `تعديل` | تعديل مقال منشور |
| `حذف` | حذف مقال |
| `أرشفة` | أرشفة مقال |
| `عاجل` | تحويل مقال إلى خبر عاجل |

### المميزات
- تجميع الرسائل المتعددة والوسائط (نافذة 30 ثانية)
- استخراج نص من الصور بالذكاء الاصطناعي (OCR - GPT-4o)
- استخراج محتوى ملفات Word
- تتبع نافذة الجلسة (24 ساعة) للامتثال
- إرسال رسائل صادرة للصحفيين

## تحسين محركات البحث SEO

### خريطة الموقع (Sitemap)
- **Sitemap عام** (`/sitemap.xml`): آخر 1,000 مقال منشور
- **Google News Sitemap** (`/sitemap-news.xml`): مقالات آخر 48 ساعة (500 مقال)
- **robots.txt**: ديناميكي مع قواعد مخصصة لـ Googlebot وBingbot
- تخزين مؤقت ذكي (5 دقائق للعام، 3 دقائق للأخبار)

### حقن البيانات الوصفية التلقائي
- **Open Graph**: `og:title`, `og:description`, `og:image` (1200x630)
- **Twitter Cards**: `summary_large_image`
- **بيانات المقال**: وقت النشر، التعديل، القسم، الوسوم
- تحميل مسبق للصور الرئيسية لتحسين LCP

### البيانات المهيكلة (JSON-LD)
| النوع | الاستخدام |
|-------|----------|
| `NewsArticle` | المقالات - عنوان، كاتب، ناشر، تواريخ |
| `LiveBlogPosting` | التغطيات المباشرة |
| `WebSite` | الصفحة الرئيسية مع SearchAction |

### الفهرسة الفورية
- **IndexNow**: إشعار Bing, Yandex, Naver فوراً عند النشر
- **Google Indexing API**: طلب فهرسة فوري من Google
- **Google Sitemap Ping**: إشعار Google عند تحديث خريطة الأخبار

### الروابط الأساسية (Canonical)
- `<link rel="canonical">` على كل صفحة
- `hreflang` للنسخ اللغوية المتعددة
- `x-default` للنسخة الأساسية

### أدوات SEO بالذكاء الاصطناعي
- توليد `metaTitle` و`metaDescription` و`keywords` تلقائياً
- تحليل SEO للمحتوى قبل النشر
- اختيار نموذج AI حسب اللغة:
  - العربية: Claude
  - الإنجليزية: GPT-4
  - الأوردو: Gemini
- توليد نص بديل (Alt Text) للصور

### Social Crawler Middleware
- كشف تلقائي لبرامج الزحف (Facebook, WhatsApp, Twitter, Google)
- تقديم HTML خفيف مع بيانات OG كاملة للروبوتات
- حل مشكلة SEO في تطبيقات الصفحة الواحدة (SPA)

## الصلاحيات والأدوار RBAC

### الأدوار المتاحة

| الدور | الوصف | المستوى |
|-------|-------|---------|
| `system_admin` | مدير النظام - وصول كامل | الأعلى |
| `admin` | مدير - إدارة المستخدمين والمحتوى | عالي |
| `editor` | محرر - إنشاء ونشر وتحرير المحتوى | متوسط-عالي |
| `reporter` | مراسل - إنشاء وتحرير مقالاته فقط | متوسط |
| `opinion_author` | كاتب رأي - كتابة مقالات الرأي | متوسط |
| `comments_moderator` | مشرف تعليقات - إدارة التعليقات | محدود |
| `media_manager` | مدير وسائط - إدارة المكتبة الإعلامية | محدود |
| `reader` | قارئ - مستخدم عادي | الأدنى |

### تفاصيل الصلاحيات

#### مدير النظام (`system_admin`)
- صلاحية شاملة (`*`) - وصول لجميع المميزات بلا استثناء

#### المدير (`admin`)
- **المستخدمون**: عرض، إنشاء، تعديل، حذف، تغيير الأدوار
- **المحتوى**: عرض، نشر، تحرير الكل، حذف
- **النظام**: عرض الإعدادات، تحديثها، عرض سجل التدقيق، إدارة الإعدادات

#### المحرر (`editor`)
- **المحتوى**: إنشاء، تحرير الكل، نشر، تمييز المقالات
- **الأقسام**: إنشاء، تحديث
- **الذكاء الاصطناعي**: توليد محتوى، توليد صور

#### المراسل (`reporter`)
- **المحتوى**: إنشاء، تحرير مقالاته فقط
- **التحليلات**: عرض تحليلاته فقط
- **الوسائط**: رفع وعرض

#### كاتب الرأي (`opinion_author`)
- **محتوى الرأي**: إنشاء، تحرير مقالاته، تقديم للمراجعة

#### مشرف التعليقات (`comments_moderator`)
- **التعليقات**: موافقة، رفض، حذف، حظر المستخدمين

### آليات التحقق من الصلاحيات
1. **Middleware قياسي**: `requireAuth`, `requirePermission`, `requireRole`
2. **حالة المدير الأعلى**: المديرون يتجاوزون فحوصات الصلاحيات الفردية
3. **استثناءات فردية**: إمكانية `السماح` أو `المنع` لصلاحية محددة لمستخدم معين
4. **التخزين المؤقت**: تخزين نتائج فحص الصلاحيات لمدة دقيقتين

## الأمان والحماية

### 1. تحديد معدل الطلبات (Rate Limiting)
| النوع | الحد | الفترة |
|-------|------|--------|
| API العام | 10,000 طلب | 15 دقيقة |
| تسجيل الدخول | 5 محاولات | 15 دقيقة |
| العمليات الحساسة | 10 طلبات | 15 دقيقة |
| عمليات الكتابة | 100 طلب | 15 دقيقة |
| رفع الوسائط | 50 عملية | 15 دقيقة |
| المتابعة | 10 طلبات | دقيقة واحدة |

### 2. رؤوس الأمان (Helmet.js)
- **سياسة أمان المحتوى (CSP)**: تكوين شامل لـ `script-src`, `connect-src`, `img-src`
- **HSTS**: مفعّل لمدة سنة مع الاشتراك المسبق (preload)
- **X-Content-Type-Options**: `noSniff` لمنع تخمين نوع المحتوى
- **XSS Filter**: فلتر هجمات البرمجة عبر الموقع
- **Referrer Policy**: سياسة إحالة آمنة

### 3. حماية CORS
- قائمة مسموحة بيئية (`ALLOWED_ORIGINS`)
- التحقق الديناميكي من المصدر
- دعم تطبيقات الجوال الأصلية (Capacitor, okhttp)
- تفعيل `credentials: true` لملفات تعريف الارتباط

### 4. حماية CSRF
- توليد رموز عشوائية (`crypto.randomBytes(32)`)
- التحقق عبر رأس `x-csrf-token`
- مقارنة آمنة زمنياً (`crypto.timingSafeEqual`)
- إعفاء: طلبات GET العامة وWebhooks

### 5. إدارة الجلسات
- **Redis**: التخزين الأساسي للجلسات (عالي الأداء)
- **PostgreSQL**: بديل احتياطي إذا لم يتوفر Redis
- **إعدادات الأمان**:
  - `httpOnly: true` - منع الوصول من JavaScript
  - `secure: true` - HTTPS فقط في الإنتاج
  - `sameSite: "lax"` - حماية من هجمات CSRF
  - مدة الجلسة: أسبوع واحد
  - `rolling: true` - تمديد تلقائي عند النشاط

### 6. المصادقة
- **كلمات المرور**: تشفير bcrypt
- **تسجيل دخول Google**: OAuth 2.0
- **تسجيل دخول Apple**: Apple Sign-In
- **المصادقة الثنائية (2FA)**: دعم التحقق بخطوتين

### 7. التحقق من المدخلات
- **Zod**: مخططات تحقق شاملة لجميع نقاط API
- **Multer**: حدود حجم الملفات (10MB) وفلترة أنواع MIME

### 8. سجلات التدقيق (Audit Logs)
- تسجيل جميع الإجراءات: المستخدم، الإجراء، نوع الكيان
- حفظ القيمة القديمة والجديدة لكل تغيير
- مراقبة الأداء: تتبع الطلبات البطيئة (>1 ثانية)
- صفحة سجلات النشاط في لوحة التحكم

### 9. أمان قاعدة البيانات
- **Drizzle ORM**: حماية ضمنية من حقن SQL
- **متغيرات البيئة**: تحميل آمن عبر dotenv
- **اتصال مشفر**: SSL/TLS لقاعدة البيانات
